A simple text stack field, with the default settings. Check out this page's source to view the code that generated it.
By setting a low idleDelay (100) and a low maxInterval (300), we get very short undo blocks. Try typing a bit, then press the undo key several times. These undo snapshots are probably too small to be useful, but this is a good demonstration of the control TextStack gives developers over user input.
This textarea has a very long idleDelay (2000) and maxInterval (10000). We basically never see undo snapshots - you'll either have to type forever, or wait a long time between typing spurts to get one.
